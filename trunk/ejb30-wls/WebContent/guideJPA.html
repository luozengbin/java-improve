You can see how EJB3 entity model works with Java Persistence Architecture through a set of examples in this page.

Run the examples
Each of these samples perform a basic operation that requires interaction of persistent entities with the underlying database, e.g.
  • start a transaction
  • query persistent instances
  • persist new instance graphs

    The samples are based on a Plain, Old Java Object (POJO) based Persistent EJB Domain Model
